# QR码游戏系统使用说明

## 系统简介
这是一个使用二维码记录摊位游戏的Web应用程序，支持管理2-30个摊位游戏和约300名用户。

## 主要文件说明

### 核心功能文件
1. **index.html** - 系统首页，提供管理员和用户登录入口
2. **admin-login.html** - 管理员登录页面（账号：admin/admin123）
3. **admin-main.html** - 管理员主控制台
4. **admin-games.html** - 游戏管理页面
5. **admin-users.html** - 用户管理页面
6. **user-login.html** - 用户登录页面（支持二维码扫描）
7. **user-dashboard.html** - 用户控制台

### 数据库工具
8. **fix-database.html** - 数据库修复工具（推荐使用）
9. **init-database.html** - 数据库初始化工具
10. **reset-database.html** - 数据库重置工具

## 正确的测试步骤

### 第一步：修复数据库
1. 打开 **fix-database.html**
2. 点击"开始修复数据库"按钮
3. 等待修复完成（约2-3分钟）

### 第二步：测试管理员功能
1. 打开 **index.html** → 点击"管理员登录"
2. 输入账号：admin，密码：admin123
3. 进入 **admin-main.html** 主控制台
4. 测试游戏管理和用户管理功能

### 第三步：测试用户功能
1. 打开 **index.html** → 点击"用户登录"
2. 使用测试用户ID：U001 登录
3. 测试游戏成绩记录功能

## 系统功能
- 游戏管理：新增、编辑、删除游戏
- 用户管理：新增、编辑、删除用户
- 成绩记录：扫描二维码记录游戏成绩
- 数据统计：查看游戏数据统计

## 技术栈
- 前端：HTML5, CSS3, JavaScript, Tailwind CSS
- 数据库：Supabase PostgreSQL
- 部署：GitHub + Vercel

## 注意事项
1. 首次使用请先运行数据库修复工具
2. 确保网络连接正常以访问Supabase数据库
3. 二维码扫描需要浏览器支持摄像头访问
