详细调试工具问题修复总结

修复时间: 2026-01-10

问题概述

详细调试工具在检查管理员账号时出现错误： Cannot read properties of undefined (reading 'substring')

同时发现数据库中管理员表的密码字段名称可能不一致，导致登录失败。

修复内容

修复管理员账号检查功能
问题：在显示管理员账号信息时，直接访问 password 字段导致空引用错误
解决方案： 添加了安全的字段检查和默认值处理，支持 password 和 password_hash 两种字段名
增强创建管理员账号功能
问题：创建管理员账号时只尝试使用 password 字段，不兼容 password_hash 字段
解决方案： 实现了智能字段检测，自动尝试两种字段名，提高兼容性
改进登录验证逻辑
问题：登录验证只检查 password 字段，不支持 password_hash 字段
解决方案： 修改验证逻辑，同时检查两个可能的密码字段，确保登录成功
优化密码更新功能
问题：密码更新功能只更新 password 字段，不兼容 password_hash 字段
解决方案： 实现了动态字段检测，自动选择正确的字段进行更新
