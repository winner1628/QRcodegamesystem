数据库表结构

QR码游戏系统使用的数据库表及其结构说明

表概览

admins

管理员用户表

存储系统管理员账号信息

users

普通用户表

存储游戏玩家账号信息

games

游戏表

存储可玩游戏的基本信息

game_records

游戏记录表

存储用户游戏记录和得分

game_statistics

游戏统计表

存储游戏统计数据和排行榜

admins 表

管理员用户表
字段名 数据类型 约束 描述
id integer PRIMARY KEY, AUTOINCREMENT 管理员ID
username varchar(50) UNIQUE, NOT NULL 管理员用户名
password varchar(255) NOT NULL 密码（Base64编码）
created_at timestamp DEFAULT CURRENT_TIMESTAMP 创建时间
updated_at timestamp NULL 更新时间
说明： 存储系统管理员账号信息，默认账号为 admin/admin123。

users 表

普通用户表
字段名 数据类型 约束 描述
id varchar(20) PRIMARY KEY 用户ID（格式：U000001）
username varchar(50) UNIQUE, NOT NULL 用户名
password varchar(255) NOT NULL 密码
role varchar(20) DEFAULT 'user' 用户角色（user/admin）
total_score integer DEFAULT 0 总得分
created_at timestamp DEFAULT CURRENT_TIMESTAMP 创建时间
updated_at timestamp NULL 更新时间
说明： 存储游戏玩家账号信息，支持普通用户和管理员角色。

games 表

游戏表
字段名 数据类型 约束 描述
id varchar(20) PRIMARY KEY 游戏ID（格式：G000001）
name varchar(100) NOT NULL 游戏名称
description text NULL 游戏描述
max_score integer DEFAULT 100 最高分数
is_active boolean DEFAULT true 是否激活
created_at timestamp DEFAULT CURRENT_TIMESTAMP 创建时间
updated_at timestamp NULL 更新时间
说明： 存储可玩游戏的基本信息，系统默认包含5个游戏。

game_records 表

游戏记录表
字段名 数据类型 约束 描述
id integer PRIMARY KEY, AUTOINCREMENT 记录ID
user_id varchar(20) NOT NULL 用户ID（外键）
game_id varchar(20) NOT NULL 游戏ID（外键）
score integer NOT NULL 游戏得分
qrcode_data text NULL QR码数据
created_at timestamp DEFAULT CURRENT_TIMESTAMP 创建时间
说明： 存储用户游戏记录和得分信息，用于追踪游戏历史。

game_statistics 表

游戏统计表
字段名 数据类型 约束 描述
id integer PRIMARY KEY, AUTOINCREMENT 统计ID
game_id varchar(20) NOT NULL 游戏ID（外键）
total_plays integer DEFAULT 0 总游玩次数
high_score integer DEFAULT 0 最高分
avg_score integer DEFAULT 0 平均分数
top_players json NULL 排行榜数据
updated_at timestamp DEFAULT CURRENT_TIMESTAMP 更新时间
说明： 存储游戏统计数据和排行榜信息，用于展示游戏成绩排名。

SQL 创建语句

创建 admins 表

CREATE TABLE IF NOT EXISTS admins (
    id SERIAL PRIMARY KEY,
    username VARCHAR(50) UNIQUE NOT NULL,
    password VARCHAR(255) NOT NULL,
    created_at TIMESTAMP DEFAULT CURRENT_TIMESTAMP,
    updated_at TIMESTAMP
);

创建 users 表

CREATE TABLE IF NOT EXISTS users (
    id VARCHAR(20) PRIMARY KEY,
    username VARCHAR(50) UNIQUE NOT NULL,
    password VARCHAR(255) NOT NULL,
    role VARCHAR(20) DEFAULT 'user',
    total_score INTEGER DEFAULT 0,
    created_at TIMESTAMP DEFAULT CURRENT_TIMESTAMP,
    updated_at TIMESTAMP
);

创建 games 表

CREATE TABLE IF NOT EXISTS games (
    id VARCHAR(20) PRIMARY KEY,
    name VARCHAR(100) NOT NULL,
    description TEXT,
    max_score INTEGER DEFAULT 100,
    is_active BOOLEAN DEFAULT true,
    created_at TIMESTAMP DEFAULT CURRENT_TIMESTAMP,
    updated_at TIMESTAMP
);

创建 game_records 表

CREATE TABLE IF NOT EXISTS game_records (
    id SERIAL PRIMARY KEY,
    user_id VARCHAR(20) NOT NULL,
    game_id VARCHAR(20) NOT NULL,
    score INTEGER NOT NULL,
    qrcode_data TEXT,
    created_at TIMESTAMP DEFAULT CURRENT_TIMESTAMP,
    FOREIGN KEY (user_id) REFERENCES users(id),
    FOREIGN KEY (game_id) REFERENCES games(id)
);

创建 game_statistics 表

CREATE TABLE IF NOT EXISTS game_statistics (
    id SERIAL PRIMARY KEY,
    game_id VARCHAR(20) NOT NULL,
    total_plays INTEGER DEFAULT 0,
    high_score INTEGER DEFAULT 0,
    avg_score INTEGER DEFAULT 0,
    top_players JSON,
    updated_at TIMESTAMP DEFAULT CURRENT_TIMESTAMP,
    FOREIGN KEY (game_id) REFERENCES games(id)
);
